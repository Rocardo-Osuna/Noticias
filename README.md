# Pagina de noticias/blog personas

Empece esta proyecto personal para practicar un poco de Django pero actualmente
no puedo continuarlo, maneja usuarios, vistas protegidas para los distintos tipos
de usuarios (vista para todos los usuarios, creacion y actualizacion de noticias
para los usuarios de tipo Editor), uso de CKEDITOR como editor de texto enriquecido, noticias con sus
distintas categorias y paginas para filtrar

## Requisitos
- Python
- Visual studio code
- Dependencias (incluidas en requirements)

## Como hacer una prueba
- De preferencia tener el proyecto en una carpeta
- Crear un entorno virtual en el mismo directorio del proyecto 'python -m venv venv'
- Entrar a la carpeta venv 'cd venv/scripts' dentro escribir 'activate'
- Entrar al directorio del proyecto
- Instalar dependencias 'pip install -r requirements.txt'
- Ejecutar el programa 'python manage.py runserver'
